# rDNA_variant_caller
Repository to call rDNA variants from short read sequencing

# Software dependencies

| Package | Version | 
| -------- | ------- | 
| MUMMER | 4.0.0rc1 | 
| ANACONDA | 2023.07 | 
| seqkit | 2.8.0 |
| Samtools | 1.19.2 |
| bwa | 0.7.17 |
| gatk | 4.5.0.0 |
| java-openjdk | 17.0.11+9|
| bcftools | 1.19 |
| R | 4.3.2 |

# Running the code

To run the code sequentially
```
./Pipeline.sh
```

To run the code in a parallel manner, check inside Scripts_parallel/
