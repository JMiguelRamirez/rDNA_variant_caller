# rDNA_variant_caller
Repository to call rDNA variants from short read sequencing

# Software dependencies

| Package | Version | 
| -------- | ------- | 
| MUMMER | 4.0.0rc1 | 
| ANACONDA | 2023.07 | 
| seqkit | 2.8.0 |


# Running the code

Code to run the code sequentially
```
./Pipeline.sh
```

Code to run the code in a parallel manner
```
./Pipeline_parallel.sh
```
